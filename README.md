# 🌾 Seeds Classification using Machine Learning

## 📌 Project Overview

This project predicts the type of wheat seed using Machine Learning based on seven physical characteristics of the seed. The model is trained using the Seeds Dataset and deployed with Streamlit.

---

## 🚀 Live Demo

🔗 https://seeds-classification-xog7bvszycnkkav2qeiuhk.streamlit.app/

---

## ✨ Features

- Predicts Wheat Seed Class
- Interactive Streamlit Web Application
- Real-Time Predictions
- User-Friendly Interface
- Machine Learning Classification

---

## 🌾 Wheat Classes

- 🌾 Class 1 → Kama Wheat
- 🌾 Class 2 → Rosa Wheat
- 🌾 Class 3 → Canadian Wheat

---

## 📊 Features Used

- Area
- Perimeter
- Compactness
- Kernel Length
- Kernel Width
- Asymmetry Coefficient
- Kernel Groove Length

---

## 🤖 Machine Learning Models

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)

**Best Model:** K-Nearest Neighbors (KNN)
